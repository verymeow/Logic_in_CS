Class Variables and Methods:Tree

This will be the stack where nodes parsed from input will be stacked. Will become final output tree.

VARIABLES

  1. Node_of_stack* top

METHODS

  1. Tree()

    Constructor

  2. void push(Tree_node* new_node)

    This pushes a new Node_of_stack onto the stack of nodes.

  3. Tree_node* pop()

    This pops out the top Node_of_stack from the stack.

  4. Tree_node* tops()

    Returns pointer to top node of current stack.

  5. void append(char elem)

    Adds the parsed element from prefix string to the tree.

  6. int evaluate()

    Function for user to evaluate expression, no parameters for method.

  7. int evaluates(Tree_node* k)
    • Method for internal computation, has a node as a method.
    • Since it eventually reaches all nodes it must run in O(n), where n is the number of nodes in the tree.
    • It traverses left side depth first till it reaches the bottom the tree after which it backtracks.
  8. int convert_to_int(char elem)

    Converts digit character to integer.

  9. void make_tree(string prefix)
    • Parses the string and makes the tree.
    • Calls append method "n" times, where n is the length of input string.
    • Since "append" for a character runs in constant time we can say that make_tree runs in O(n).
  10. void infix()

    Gets the infix expression.

  11. void in_ord_trav(Tree_node* m)
    • Traverses the tree in order to get the infix expression.
    • It runs in O(n) since each node needs to be traversed.
  12. int height()

    User friendly method for height.

  13. int heights(Tree_node*a)
    • Calculates height of tree recursively
    • Recursive call goes log(n) deep where n is the no. of nodes.
