Space Complexity

  • All the data structures that are used for computations are declared globally at the start.
  • Their size is linear in the no. of characters in input expressions.
  • Important global variables are:
    1. infix[100] : contains initial input.
    2. answer[100] : contains final answer.
    3. stack_arr[100] : contains stack used for computing the prefix.
    4. cnf_exp : contains a string obtained by in-order traversal of cnf tree, minus the brackets.
  • All functions used for CNF computation use a tree of original expression, which they then modify by adding a node or two. Their space complexity is essentially linear in size of input expression.
